# Product Requirements Document

Project Name:
AI Financial Insights Dashboard

Goal:
Build a modern financial insights web application where users can explore transaction data, analyze spending patterns, detect recurring payments, and receive AI-generated financial insights.

Tech Stack:
Next.js
Supabase
Tailwind CSS
Vercel

Core Features:

User Authentication
- Sign up and login using Supabase Auth

Transaction Dashboard
- List of financial transactions
- Filter by category, date, merchant

Spending Analytics
- Charts for category spending
- Monthly spending summary

Recurring Payments Detection
- Detect subscriptions from transaction data

AI Financial Insights
- Generate insights about spending behavior

UI Requirements
- Responsive dashboard
- Smooth animations
- Minimalist fintech style

Data Source
- Use seeded transaction data stored in Supabase

Deployment
- Must deploy cleanly to Vercel
